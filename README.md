# create-midas-front

Project initializer for React and Next.js projects.

## Usage

npm:

```sh
npm create midas-front
```

pnpm:

```sh
pnpm create midas-front
```

bun:

```sh
bun create midas-front
```

yarn:

```sh
yarn create midas-front
```

## CLI Options

Use `-h` or `--help` to see all available options.
